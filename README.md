# 100 Days of Code Python Challenge Course

## Description
This is all the code created as part of Angela Yu's 100-Day Python Course on Udemy.

## Remarks
Here are a few things to point out about this repository:
* The course recommends the use of the PyCharm IDE. However I chose to use VSCode to gain familiarity with a more professional working IDE and also because I really like to work with the terminal for Git/GitHub operations.
* The python version used to produce these projects is Python v3.13.14.

## Project Documentation
Here is some brief information about the projects I created within this course. For more information look inside the main.py file of each project.
